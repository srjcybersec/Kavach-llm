# UI screenshots for Kavach.LLM

Place PNG files here so the root [README.md](../../README.md) image links resolve.

## Filenames

| File | Page |
|------|------|
| `dashboard.png` | Home — charts, summary, live threat feed, compliance PDF action |
| `playground.png` | Prompt scan cards, demo flow, optional bypass |
| `policies.png` | Rule builder, drag-and-drop order, compliance toggle |
| `audit.png` | Filters, table, export, detail slide-over |
| `settings.png` | API keys, reputation, model/webhook/team preferences |

## How to capture

1. Run `npm run dev` or bring up Docker Compose and open `http://localhost:5173`.
2. Use a consistent viewport (e.g. 1440×900) for a clean set.
3. Redact any real emails, keys, or internal URLs before committing images.

Until these files exist, the README will show broken image placeholders in some viewers; that is expected until you add the assets.
